# KodNest Premium Build System (Design System Only)

This project defines the **KodNest Premium Build System** design system and layout shell.  
It is intentionally limited to visual structure and components only—no product features have been added.

## Stack

- React 18
- Vite
- TypeScript

## Running locally

```bash
npm install
npm run dev
```

Then open the printed local URL in your browser.

## Design principles encoded

- Calm, intentional, coherent, confident
- No gradients, glassmorphism, neon, or playful motion
- Strict spacing scale: 8 / 16 / 24 / 40 / 64
- Serif headings, clean sans-serif body, generous line height
- Global layout: Top Bar → Context Header → Primary Workspace + Secondary Panel → Proof Footer

Use this as the foundation for premium B2C SaaS experiences built on top of KodNest.
